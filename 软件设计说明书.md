## 1. 系统概述

### 1.1 项目背景

宁波乐科科信息技术有限公司（Rococo）是一家以"心"为核心理念的日本IT服务企业Rococo集团的中国子公司。公司致力于为企业提供数字化转型服务，包括IT咨询、定制开发、低代码平台开发、系统重构等服务。

### 1.2 项目目标

| 目标 | 说明 |
|------|------|
| 品牌形象展示 | 通过官网展示公司形象、服务能力和成功案例 |
| 中日双语支持 | 支持日语(/ja/)和中文(/zh/)两种语言 |
| 内容后台管理 | 通过Strapi CMS管理网站内容，支持实时更新 |
| SEO友好 | 支持搜索引擎抓取和AI内容理解 |
| 响应式设计 | 适配PC端和移动端 |

### 1.3 目标用户

| 用户类型 | 需求 |
|---------|------|
| 潜在客户 | 了解公司业务、服务能力、联系方式 |
| 求职者 | 查看招聘信息、了解公司背景 |
| 合作伙伴 | 了解公司资质、服务内容 |
| 内部员工 | 内容管理、日常维护 |

## 4. 核心组件设计

### 4.1 核心类/接口设计

#### 4.1.1 useLocale Composable

```typescript
// composables/useLocale.ts
type Locale = 'ja' | 'zh'
type LocaleMessages = Record<string, any>

export function useLocale() {
  // 语言状态
  const locale = useState<Locale>('locale', () => 'ja')

  // 计算属性
  const isJapanese = computed(() => locale.value === 'ja')
  const isChinese = computed(() => locale.value === 'zh')

  // 设置语言
  function setLocale(newLocale: Locale): void

  // 切换语言
  function toggleLocale(): void

  // 翻译函数
  function t(key: string): string

  return {
    locale,       // 'ja' | 'zh'
    isJapanese,   // boolean
    isChinese,    // boolean
    setLocale,   // (locale: Locale) => void
    toggleLocale,// () => void
    t,           // (key: string) => string
  }
}
```

#### 4.1.2 useStrapi Composable

```typescript
// composables/useStrapi.ts
interface NewsItem {
  id: number
  title: string
  date: string
  excerpt: string
  content?: string
}

interface CaseStudy {
  id: number
  title: string
  industry: string
  image?: string
  content?: string
}

interface Config {
  phone: string
  email: string
  address: { ja: string; zh: string }
  wechat: string
  salesWechat: string
  icp: { ja: string; zh: string }
}

interface ContactForm {
  name: string
  company?: string
  email: string
  phone?: string
  message: string
  source: string
}

export const useStrapi = () => {
  // 获取新闻列表
  async function fetchNews(params?: {
    page?: number
    pageSize?: number
    filters?: Record<string, any>
  }): Promise<{ data: NewsItem[]; meta?: any }>

  // 获取单条新闻
  async function fetchNewsById(id: number | string): Promise<{ data: NewsItem | null }>

  // 获取案例列表
  async function fetchCaseStudies(params?: {
    page?: number
    pageSize?: number
    filters?: Record<string, any>
  }): Promise<{ data: CaseStudy[]; meta?: any }>

  // 获取单条案例
  async function fetchCaseStudyById(id: number | string): Promise<{ data: CaseStudy | null }>

  // 获取网站配置
  async function fetchConfig(): Promise<{ data: Config }>

  // 提交联系表单
  async function submitContact(data: ContactForm): Promise<{ success: boolean }>

  return {
    fetchNews,
    fetchNewsById,
    fetchCaseStudies,
    fetchCaseStudyById,
    fetchConfig,
    submitContact,
  }
}
```
